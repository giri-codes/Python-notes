# Python Booleans

This project contains simple Python examples to understand Boolean values
(`True` and `False`) and how they are used in decision making.

## What is covered
- Boolean values
- Comparison operators
- Logical operators (and, or, not)
- Booleans with if conditions
- Simple real-life examples

## Purpose
To practice Python Booleans using easy and real-life based code examples.
